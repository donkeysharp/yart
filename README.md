React Template
==============

Yet another npm-scripts-based SASS, ES2015(Babel), Browserify, React/Redux, React-Router SPA template

####Requirements
* Install nodejs
* Install rvm and some ruby version
* Install gem sass

####Options
For production run:

    $ npm run build

For development run:

    $ npm run watch


Browse *localhost:8000* to test
